# LinkedIn Post Agent

AI agent that generates and posts professional LinkedIn content using OpenAI, LangChain, and MCP architecture.

## Features

- Human-style LinkedIn post generation
- LangChain agent with OpenAI GPT-4o
- Modular MCP design
- LinkedIn API integration
- Streamlit UI + FastAPI backend

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Configure environment variables in `.env`

## Running

**Backend:**
```bash
cd backend
python main.py
```

**Frontend:**
```bash
cd frontend
streamlit run app.py
```
